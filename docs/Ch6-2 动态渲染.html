Ch6-2 动态渲染（未完成）

动态渲染（dynamic rendering），简而言之是让渲染通道和帧缓冲一起滚蛋的渲染方式。
所谓动态，指渲染相关的参数大都在渲染开始时指定。

不需要渲染通道和帧缓冲，意味着省去了对它们进行管理的麻烦！尤其是，不再需要“因窗口大小改变而在重建交换链后重建渲染通道和帧缓冲”。
渲染通道的子通道，开始和结束时会执行子通道依赖，并进行图像内存布局转换。
而动态渲染的“开始”和“结束”只是开始和结束，其间就只是一次独立的渲染，子通道依赖的作用和图像内存布局转换得通过图像内存屏障来实行。
往下阅读之前，请确保已经大致了解了图像内存屏障的用法。

本节在之前Ch2中绘制三角形代码的基础上，使用动态渲染的方法来进行绘制，步骤如下：
1.开启设备特性
2.更改管线创建信息
3.定义动态渲染前后的内存屏障
4.调用命令，开始/结束动态渲染

开启动态渲染设备特性

无图像帧缓冲在Vulkan1.3版本之前由设备级扩展"VK_KHR_dynamic_rendering"提供（且依赖于另其他扩展），从1.3版本开始为Vulkan的核心功能。

类似前一节的无图像帧缓冲，1.0下的开启方法太麻烦不谈。Vulkan版本达到1.3的话，设备特性已在graphicsBase::CreateDevice(...)中“能开尽开”，主函数中只检查设备特性是否支持： 

int main() {
    graphicsBase::Base().UseLatestApiVersion();
    if (graphicsBase::Base().ApiVersion() < VK_API_VERSION_1_1)
        return -1;
    if (graphicsBase::Base().ApiVersion() < VK_API_VERSION_1_3) {
        /*待填充*/
    }
    else
        if (!InitializeWindow({ 1280, 720 }) ||
            !graphicsBase::Base().PhysicalDeviceVulkan13Features().dynamicRendering)
            return -1;

    /*...*/
}

若Vulkan版本达到1.1但未到1.3，使用以下代码通过扩展开启： 

int main() {
    graphicsBase::Base().UseLatestApiVersion();
    if (graphicsBase::Base().ApiVersion() < VK_API_VERSION_1_1)
        return -1;
    if (graphicsBase::Base().ApiVersion() < VK_API_VERSION_1_3) {
        if (graphicsBase::Base().ApiVersion() < VK_API_VERSION_1_2)
            graphicsBase::Base().AddDeviceExtension(VK_KHR_CREATE_RENDERPASS_2_EXTENSION_NAME),  //在1.2版本中进入了核心功能
            graphicsBase::Base().AddDeviceExtension(VK_KHR_DEPTH_STENCIL_RESOLVE_EXTENSION_NAME);//在1.2版本中进入了核心功能
        graphicsBase::Base().AddDeviceExtension(VK_KHR_DYNAMIC_RENDERING_EXTENSION_NAME);
        VkPhysicalDeviceDynamicRenderingFeatures physicalDeviceDynamicRenderingFeatures = {
            VK_STRUCTURE_TYPE_PHYSICAL_DEVICE_DYNAMIC_RENDERING_FEATURES,
        };
        graphicsBase::Base().AddNextStructure_PhysicalDeviceFeatures(physicalDeviceDynamicRenderingFeatures);
        if (!InitializeWindow({ 1280, 720 }) ||
            !physicalDeviceDynamicRenderingFeatures.dynamicRendering)
            return -1;
    }
    else
        if (!InitializeWindow({ 1280, 720 }) ||
            !graphicsBase::Base().PhysicalDeviceVulkan13Features().dynamicRendering)
            return -1;

    /*...*/
}
